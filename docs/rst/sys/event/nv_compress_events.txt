nv_compress_events


 NAME:
	nv_compress_events


 PURPOSE:
	Returns only unique events from the given buffer.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	events = nv_compress_events(buf)


 ARGUMENTS:
  INPUT:
	buf:	Array of nv_event_struct.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	All events in buf that are unique.


 STATUS:
	Complete


 MODIFICATION HISTORY:
 	Written by:	Spitale
