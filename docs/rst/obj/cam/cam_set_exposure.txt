cam_set_exposure


 NAME:
	cam_set_exposure


 PURPOSE:
       Replaces the exposure duration for each given camera descriptor.


 CATEGORY:
	NV/LIB/CAM


 CALLING SEQUENCE:
	cam_set_exposure,cd,exposure


 ARGUMENTS:
  INPUT:
	cd:	     Array (nt) of CAMERA descriptors.

       exposure:    Array (nt) of new exposure values.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:  NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
