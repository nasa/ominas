pnt_template


 NAME:
	pnt_template


 PURPOSE:
	Creates a new POINT using an existing one as a template.


 CATEGORY:
	NV/OBJ/PNT


 CALLING SEQUENCE:
	ptd = pnt_template(ptd0)


 ARGUMENTS:
  INPUT:
	ptd0:		POINT object.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	New POINT with array fields left blank.


 STATUS:
	Complete




 MODIFICATION HISTORY:
 	Written:	Spitale, 11/2015
