dsk_il


 NAME:
	dsk_il


 PURPOSE:
	Returns il for each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	il = dsk_il(dkd)


 ARGUMENTS:
  INPUT: NONE
	dkd:	 Any subclass of DISK.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:

  OUTPUT: NONE


 RETURN:
	il value associated with each given disk descriptor.


 STATUS:
	Complete
