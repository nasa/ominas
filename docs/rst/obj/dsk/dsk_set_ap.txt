dsk_set_ap


 NAME:
	dsk_set_ap


 PURPOSE:
	Replaces ap in each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	dsk_set_ap, bx, ap


 ARGUMENTS:
  INPUT:
	dkd:	 Array (nt) of any subclass of DISK.

	ap:	 New ap value.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
