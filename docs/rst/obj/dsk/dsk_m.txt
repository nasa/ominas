dsk_m


 NAME:
	dsk_m


 PURPOSE:
	Returns m for each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	m = dsk_m(dkd)


 ARGUMENTS:
  INPUT: NONE
	dkd:	 Any subclass of DISK.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:

  OUTPUT: NONE


 RETURN:
	m value associated with each given disk descriptor.


 STATUS:
	Complete
