dsk_set_l


 NAME:
	dsk_set_l


 PURPOSE:
	Replaces l in each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	dsk_set_l, bx, l


 ARGUMENTS:
  INPUT:
	dkd:	 Array (nt) of any subclass of DISK.

	l:	 New l value.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
