dsk_nm


 NAME:
	dsk_nm


 PURPOSE:
	Returns nm for each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	nm = dsk_nm(dkd)


 ARGUMENTS:
  INPUT: NONE
	dkd:	 Any subclass of DISK.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:

  OUTPUT: NONE


 RETURN:
	nm value associated with each given disk descriptor.


 STATUS:
	Complete
