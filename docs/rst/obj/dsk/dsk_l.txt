dsk_l


 NAME:
	dsk_l


 PURPOSE:
	Returns l for each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	l = dsk_l(dkd)


 ARGUMENTS:
  INPUT: NONE
	dkd:	 Any subclass of DISK.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:

  OUTPUT: NONE


 RETURN:
	l value associated with each given disk descriptor.


 STATUS:
	Complete
