dsk_set_m


 NAME:
	dsk_set_m


 PURPOSE:
	Replaces m in each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	dsk_set_m, bx, m


 ARGUMENTS:
  INPUT:
	dkd:	 Array (nt) of any subclass of DISK.

	m:	 New m value.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
