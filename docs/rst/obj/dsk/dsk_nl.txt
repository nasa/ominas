dsk_nl


 NAME:
	dsk_nl


 PURPOSE:
	Returns nl for each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	nl = dsk_nl(dkd)


 ARGUMENTS:
  INPUT: NONE
	dkd:	 Any subclass of DISK.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:

  OUTPUT: NONE


 RETURN:
	nl value associated with each given disk descriptor.


 STATUS:
	Complete
