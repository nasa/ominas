dsk_set_em


 NAME:
	dsk_set_em


 PURPOSE:
	Replaces em in each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	dsk_set_em, bx, em


 ARGUMENTS:
  INPUT:
	dkd:	 Array (nt) of any subclass of DISK.

	em:	 New em value.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
