dsk_set_nl


 NAME:
	dsk_set_nl


 PURPOSE:
	Replaces nl in each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	dsk_set_nl, bx, nl


 ARGUMENTS:
  INPUT:
	dkd:	 Array (nt) of any subclass of DISK.

	nl:	 New nl value.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
