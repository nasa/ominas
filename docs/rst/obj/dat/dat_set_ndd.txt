dat_set_ndd


 NAME:
	dat_set_ndd


 PURPOSE:

	NV/SYS


 CALLING SEQUENCE:
	dat_set_ndd, ndd


 ARGUMENTS:
  INPUT:
	ndd:	New ndd value.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
