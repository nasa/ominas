dh_validate


 NAME:
	dh_validate


 PURPOSE:
	Validates a detached header.


 CATEGORY:
	DAT/DH


 CALLING SEQUENCE:
	result = dh_validate(dh)


 ARGUMENTS:
  INPUT:
	dh:		String array containing the detached header.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	1 if valid, 0 otherwise.


 STATUS:
	Complete


 SEE ALSO:
	dh_write
