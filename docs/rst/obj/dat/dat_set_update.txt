dat_set_update


 NAME:
	dat_set_update


 PURPOSE:
	Changes the update flag in a data descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	dat_set_update, dd, update


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor.

	update:	New update flag.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
