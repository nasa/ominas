dat_htype


 NAME:
	dat_htype


 PURPOSE:
	Returns the header type associated with a data descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	htype = dat_htype(dd)


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	String giving the header type.


 STATUS:
	Complete


 MODIFICATION HISTORY:
 	Written by:	Spitale, 9/2017
