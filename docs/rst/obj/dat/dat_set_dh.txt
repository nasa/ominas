dat_set_dh


 NAME:
	dat_set_dh


 PURPOSE:
	Replaces the dh value in a data descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	dat_set_dh, dd, dh


 ARGUMENTS:
  INPUT:
	dd:		Data descriptor.

	dh:	New dh value.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
