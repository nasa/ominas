dat_typecode


 NAME:
	dat_typecode


 PURPOSE:
	Returns the type code associated with a data descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	typecode = dat_typecode(dd)


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	Integer giving the typecode.


 STATUS: Complete
