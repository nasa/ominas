dat_set_dim


 NAME:
	dat_set_dim


 PURPOSE:
	Replaces the dimensions in a data descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	dat_set_dim, dd, dim


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor.

	dim:	New dimensions.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
