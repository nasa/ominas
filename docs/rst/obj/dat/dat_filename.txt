dat_filename


 NAME:
	dat_filename


 PURPOSE:
	Returns the file name associated with a data descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	filename = dat_filename(dd)


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	String giving the filename.


 STATUS:
	Complete
