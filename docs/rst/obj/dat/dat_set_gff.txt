dat_set_gff


 NAME:
	dat_set_gff


 PURPOSE:
	Replaces the gff value in a data descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	dat_set_gff, dd, gff


 ARGUMENTS:
  INPUT:
	dd:		Data descriptor.

	gff:	New gff value.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
