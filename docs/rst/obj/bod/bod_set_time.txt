bod_set_time


 NAME:
	bod_set_time


 PURPOSE:
	Replaces the time of each given body descriptor.


 CATEGORY:
	NV/LIB/BOD


 CALLING SEQUENCE:
	bod_set_time, bx, time


 ARGUMENTS:
  INPUT:
	bx:	 Any subclass of BODY.

	time:	 New time value.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
