bod_ib_data


 NAME:
	bod_ib_data


 PURPOSE:
	Returns the function data for a body descriptor.


 CATEGORY:
	NV/LIB/BOD


 CALLING SEQUENCE:
	data = bod_ib_data(bd)


 ARGUMENTS:
  INPUT: NONE
	bd:	 Body descriptor.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:

  OUTPUT: NONE


 RETURN:
	Function data associated with the given body descriptor.


 STATUS:
	Complete

 	Adapted by:	Spitale, 7/2016
