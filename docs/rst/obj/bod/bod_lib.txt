bod_lib


 NAME:
	bod_lib


 PURPOSE:
       Returns the phase of the libration vector at body time for
       each given body descriptor.


 CATEGORY:
	NV/LIB/BOD


 CALLING SEQUENCE:
	lib = bod_lib(bx)


 ARGUMENTS:
  INPUT: NONE
	bx:	 Array (nt) of any subclass of BODY descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:

  OUTPUT: NONE


 RETURN:
       Phase of the libraton vectors at body time associated with
       each given body descriptor.


 STATUS:
	Complete
